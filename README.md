# API F1 Web Flux

API com Spring WebFlux que utiliza o endpoint da ergast para consumir os dados de F1 e salvar em base para posterior consulta.  

## Estrutura do Projeto

```
DDD
Spring Web Flux
Spring Cloud Feign
Hystrix
JUnit
Mockito
MapStruct
Spring Data 
Exception Handler
```
